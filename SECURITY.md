# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT open a public issue.**
2. Go to the [Security Advisories](https://github.com/GeiserX/duplicacy-ha/security/advisories) page.
3. Click **"Report a vulnerability"** and fill in the details.

You can expect an initial response within 72 hours. Confirmed vulnerabilities will be patched as quickly as possible and disclosed once a fix is available.
