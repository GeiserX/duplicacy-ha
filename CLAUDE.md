# CLAUDE.md — duplicacy-ha

## Overview
Home Assistant custom integration for monitoring Duplicacy backups. Connects to the duplicacy-exporter Prometheus endpoint to surface backup status, progress, speed, and historical metrics as HA entities.

## Tech Stack
- Python (Home Assistant custom component)
- HACS-compatible integration
- pytest for testing
- Codecov for coverage
- GitHub Actions CI

## Development
```bash
pip install -r requirements-test.txt
pytest tests/
```

Integration is installed via HACS or by copying `custom_components/duplicacy/` into the HA config directory.

## Architecture
- `custom_components/duplicacy/` — HA integration code
- `tests/` — test suite
- `hacs.json` — HACS metadata
- `docs/` — documentation and images

## Key Rules
- Requires a running duplicacy-exporter instance exposing `/metrics` and `/health`
- Minimum Home Assistant version: 2024.1
- Creates separate HA devices per backup job (snapshot ID + storage target)
- Never hardcode exporter URLs; configured via HA config flow
